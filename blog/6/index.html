iPhone XR 完美越狱 成功实例

发表于2021-10-31|更新于2022-04-17|越狱
|字数总计:2.4k|阅读时长:7分钟

完美越狱
最大的优势，激活一次，就是越狱状态，重启后设备不丢失越狱环境，随意开关机；
简单，没那么多麻烦

不完美越狱
重启后设备恢复到未越狱环境，需要使用工具重新激活，才能进入越狱环境
不完美越狱，需要考虑越狱工具安装和持续可用性

越狱支持

本次越狱支持的处理器：A12 至 A14
以及越狱支持的系统版本：14.3 至 14.5.1

准备工具

所有软件安装路径不建议修改

  • 待越狱设备（iPhone或iPad等）

  • 苹果数据线（USB-Lightning）

  • Windows10以上系统（我使用的是Win10）

    也可以使用Mac等电脑（系统10.14.4+以上）

    注意：Win7系统需要装黑苹果虚拟机

  • 电脑下载 AltStore
    官网地址：https://altstore.io/

    阿里云盘下载：https://www.aliyundrive.com/s/tMfvjQrSEU1

  • 下载后，双击打开 setup.exe 文件

  • 点击 Next 下一步
  • 安装路径不改，继续点击 Next 进行下一步
  • 继续点击 Next 进行下一步
  • 安装完成后，点击 Close 即可

  • 电脑下载 iCloud
    官网地址：https://www.icloud.com/

    阿里云盘下载：https://www.aliyundrive.com/s/Upfbe3nRmxK

  • 下载好iCloud安装包，双击打开

  • 勾选 我接受许可协议中的条款，并点击 安装

  • 安装期间可能会弹出系统提示，点击 确认 或者 是 即可

  • 等待安装完成后点击 结束

  • 接着会弹出个重启提示框，点击 是 重启即可

  • 找到 iCloud 并打开，登录你的 苹果id，点击 登录
  • 将 iCloud云盘 勾选，其他两项 取消勾选，点击 应用 即可

安装iCloud后，必须要重启

  • 电脑下载 iTunes
    官网地址：https://www.apple.com.cn/itunes/

    阿里云盘下载：https://www.aliyundrive.com/s/nRodCAgAZKN

  • 下载好iTunes安装包，双击打开，点击 下一步

  • 安装路径不可更改，点击 安装

  • 安装期间可能会弹出系统提示，点击 确认 或者 是 即可

  • 电脑下载 爱思助手
    官网地址：https://www.i4.cn/

    阿里云盘下载：https://www.aliyundrive.com/s/TVLbeAkDasN

  • 按照提示安装即可，安装路径尽量不修改

  • 手机下载 unc0ver
    官网地址：https://www.unc0ver.dev/

  • 提前使用手机自带 Safari浏览器 下载

越狱步骤

实例设备：iPhone XR A12
实例系统：iOS14.4 美版

登录软件 ①

  • 电脑打开爱思，手机使用数据线连接到电脑，并信任
  • 电脑打开iCloud并登录；同理，iTunes打开并登录

    如手机无弹出信任窗口，请检查是否已连接，若无则打开爱思再次插机即可

设置WiFi同步 ②

  • 手机以及电脑连上同一个网络（完成越狱前不可切换或断开）
  • 打开iTunes，找到 通过Wi-Fi与此iPhone同步 选项，打钩，并点右下角 同步

安装AltStore ③

  • 电脑找到AltServer并右键以管理员身份运行
  • 电脑右下角找到Alt，左键选择 lnstall AltStore,再选择你的设备
  • 接下来会弹出登录的提示框，登录你手机的ID，再点lnstall
  • 第一次登录会有验证码，输入即可；再出现弹框点击 确认 即可

导入自签 ④

  • AltStore 安装到手机后，打开 Safari浏览器，找到从 unc0ver 官网下载的最新版本
  • 右上角找到ipa安装包，点击打开；分享方式导入 AltStore
  • 导入后就会开始自签，然后会弹出登录界面，登录你的id即可
  • 自签过程弹窗询问，是否安装 untether，点击 Install Untethered Jailbreak 安装

运行Fugu14 ⑤

  • 安装完成后，点击 Open Placeholder，就会第一次打开 Fugu14
  • 点击 Setup Fugu14 运行，运行结束弹窗 Setup Done 点击OK

二次运行Fugu14 ⑥

  • 切换后台返回 AltStore，AltStore 会继续自动操作，等待进度条走完，然后点击底部 Open Patchholder 按钮
  • 这时候会第二次打开Fugu14，中间的按钮显示为 Install Untether，点击这个按钮进行安装
  • 耐心等待安装，安装完成会弹窗Reboot required，点击 Reboot now 重启设备

注意：
有部分会遇到第二次进入Fugu14会白（黑）屏，耐心等待一小会即可
如若遇到长时间黑白屏，请看下面的 报错案例

重启后自动化安装 ⑦

  • 重启开机后，打开 AltStore，自动化会继续，打开就会提示要安装 unc0ver，点击 Install unc0ver 安装
  • 耐心等待 unc0ver 安装完成，安装完成底部是 Open unc0ver 按钮，点击就可以打开unc0ver，点击底部 Jailbreak 就可以进行越狱了

unc0ver越狱 ⑧

  • 点击 Jailbreak 越狱中，第一次使用unc0ver越狱会弹出个提示框，一共有三个选项，选择最下面的选项即可
  • 再快结束时，会遇到Unveil Security App的广告，只需点击左上角关闭即可
  • 越狱结束后，弹窗选择 OK 结束越狱，系统会自动重启

越狱成功

  • 正确的操作后，桌面会有两个一样的图标，一个名称是 unc0ver，一个名称是 Untether
  • 这个 Untether 在未越狱状态下可以打开使用，不会过期，无需签名
  • 当设备重启后，只需打开 Untether，点击 Jailbreak，就可以激活越狱了，直接解决了签名问题

版本升级

正常升级操作步骤为方法1
升级遇到错误请检查充电线或者网络问题

方法1出错解决不了可尝试方法2的操作流程

  • 手机连接到电脑

  • 手机重启恢复至 未越狱状态

  • 打开 Safari浏览器

  • 进入unc0ver官网（unc0ver）

  • 下载 unc0ver 最新版本，下载完成打开

  • 右上角点击分享按钮，选择Altstore导入

  • 它就会自动签名安装，等待完成后

  • 打开unc0ver，点击 Jaibroken 越狱即可

  • 手机处于越狱状态

  • 打开越狱商店Cydia，下载 ReProvision 等签名工具

  • 打开手机Safari浏览器

  • 进入unc0ver官网（unc0ver）下载最新安装包

  • 打开ReProvision签名工具登录后，选择左上方Add按钮，选择最新安装包进行签名

  • 签名后会自动安装，后续只需信任描述文件即可

报错案例

iCloud Not Found

解决方法：

  • 先找到 iCloud 安装后路径，在桌面左下角菜单中找到 iCloud右键更多，找到定位位置
  • 找到 iCloud 右键打开文件所有的位置
  • 看到安装路径 C:\Program Files (x86)\Common Files\Apple
  • 再次打开AltServer签名工具弹出 iCloud Not Found 提示框，选择 ChooseFolder 按钮，再选择 iCloud 安装路径就行，选到 Apple 就点确认
  • 再次打开AltServer签名工具就不会弹了

lnstallation failed

解决方法：

  • 使用另外一部手机开启热点，给电脑以及待越狱设备连上

Could not Find Altserver

解决方法1：

  • 手机与电脑连接曾中断过，需重新尝试

解决方法2：

  • 电脑与手机不是同一个WiFi，连接同一个即可

lnstall unc0ver

解决方法1：

  • 等待十几秒后，依旧卡住，可更换USB插口重试

解决方法2：

  • 使用爱思个人签名，再安装到手机上即可

解决方法2：

  • 关闭电脑安全软件，如Win自带防火墙拦截、电脑管家、360等安全软件

Fugu14黑屏不运行

解决方法1：
手机强制关机重启，重新越狱

解决方法2：
平刷（重置手机）清除越狱状态，重新越狱

unc0ver显示unsupported

解决方法1：

  • 关机重启尝试；unsupported 显示为 Jailbreak 即可

解决方法2：

  • 重新安装AltStore

unc0ver越狱后不重启

解决方法：

  • 卸载unc0ver重装即可

最后总结

其实在越狱后，完美和不完美越狱区别不大，一样的使用
本次的完美越狱，只能称为半完美越狱，重启后恢复未越狱状态+一个永不过期的越狱工具

解决了证书掉签的烦恼，可以随便开关机
即使装错了插件，也可以进入安全模式

比以往的一键越狱多了两个步骤
除了从电脑安装AltStore到手机这个步骤容易出错
（大部分是因为网速的原因，手机AltStore和电脑AltServer之间需要传输，速度一慢或者停下来，都可能安装不上，导致一直没反应）

后面的只要按照步骤进行，百分之九十八都可以正常越狱

再次提醒：
整个过程线不可以中断，容易出错报错

自家电脑安装iTunes以及iCloud后，电脑需要关机重启

注意事项

  • 截本文前，Fugu14 越狱，需要使用电脑。只是这次 AltStore 自动化安装 Fugu14，不需要使用 macOS 自己编译安装了
  • AltStore 1.4.8 版本才有内置的 Fugu14
  • AltStore 的安装和使用，需要一定技巧，或许会遇到一些问题，需要自己摸索解决
  • 请仔细检查自己的设备和系统，是否在本次越狱支持范围之内，确定后才动手
  • 资料无价。在你动手之前，请做好重要资料的备份，以免意外情况造成资料丢失
  • 截本文前，文章介绍的仅仅是目前的情况，后续或许有变化，例如支持更多的系统与设备，例如实现真正的重启无需激活的完美越狱
